# Contributing to MECO Project

Thank you for showing interest in contributing to MECO!  
The project is still في مرحلة البناء التدريجي، والمساهمات البسيطة مرحب بها دائماً.

---

## Ways to Contribute
### ✔ Code Contributions
- إصلاحات أخطاء  
- تحسينات في الواجهة  
- تحسين تجربة MECO Wallet Web  

### ✔ Documentation
- تحسين الشروحات  
- إضافة أمثلة  
- ترجمة الملفات  

### ✔ Feedback
- اقتراح مميزات  
- الإبلاغ عن مشاكل  
- أفكار لتحسين الرؤية  

---

## Development Guide
1. قم بعمل Fork للمستودع  
2. أنشئ فرع جديد  
3. أرسل التعديلات  
4. افتح Pull Request  

---

## Rules
- حافظ على بساطة الكود  
- التزم بتنسيق الملفات  
- لا تضف مكتبات ثقيلة غير ضرورية  

---

## Contact
للاستفسارات أو الاقتراحات، الرجاء فتح Issue داخل GitHub.

شكراً لكونك جزءًا من مشروع MECO ❤️
